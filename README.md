# Content

A PyTorch implementation of UAD, a unified framework for zero-shot anomaly detection (ZSAD) that identifies and localizes anomalies in unseen target domains without requiring any target-domain training data. 
UAD jointly models semantic regularity and structural deviations using coherent semantic contexts, prompt concatenation, and intensity-guided diffusion-based anomaly synthesis to achieve robust cross-domain generalization. 
The repository includes training and evaluation scripts, pretrained models, and instructions to reproduce results on industrial and medical anomaly datasets.
